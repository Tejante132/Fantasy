    $(document).ready(function () {
        var $searchContainer = $('#header-search-container');
        var $searchTrigger = $('#header-search-trigger');
        var $searchInput = $('#header-search-input');
        var $searchIcon = $searchTrigger.find('.icon');
        var isSearching = false; // Flag to track if a search is in progress

        $searchTrigger.on('click', function (e) {
            e.preventDefault();
            if (isSearching) { // If a search is in progress, do nothing
                return;
            }
            if (!$searchContainer.hasClass('search-active')) {
                $searchContainer.addClass('search-active');
                if (!$searchInput.prop('disabled')) { // Only focus if not disabled
                    $searchInput.focus();
                }
            }
        });

        function performSearch() {
            if (isSearching) { // If a search is already in progress, do nothing
                console.log('Search already in progress, ignoring.');
                return;
            }
            var searchTerm = $searchInput.val().trim();
            if (searchTerm) {
                isSearching = true; // Set the flag to indicate search has started
                $searchInput.prop('disabled', true); // Disable the input field
                
                // Disable search icon and trigger to prevent further clicks and provide visual feedback
                $searchIcon.css({'pointer-events': 'none', 'opacity': '0.5'});
                $searchTrigger.css({'pointer-events': 'none', 'opacity': '0.5'});

                console.log('Redirecting to search for:', searchTerm);
                window.location.href = '/Servicos?search=' + encodeURIComponent(searchTerm);
            } else {
                console.log('Empty search term, ignoring.');
            }
        }

        $searchIcon.on('click', function(e) {
            e.preventDefault(); 
            e.stopPropagation(); 

            if (isSearching) {
                return; 
            }
            if ($searchContainer.hasClass('search-active')) {
                performSearch();
            }
        });

        $searchInput.on('keypress', function (e) {
            if (e.which === 13) { // Enter key pressed
                e.preventDefault(); 
                if (isSearching) { 
                    return; 
                }
                performSearch();
            }
        });

        $(document).on('click', function(event) {
          if (isSearching) { // If a search is in progress, don't mess with the UI by closing the search bar
              return;
          }
          if (!$(event.target).closest('#header-search-container').length) {
            if ($searchContainer.hasClass('search-active')) {
              $searchContainer.removeClass('search-active');
              if (!$searchInput.prop('disabled')) { // Only clear input if not disabled (i.e., search not fired yet)
                  $searchInput.val(''); 
              }
            }
          }
        });
    });

 <li>o nome que pretende aprovar&nbsp;(consulte as&nbsp;<a href="/Empresas/Regras-de-composicao-do-nome-da-empresa" target="_blank">regras de composição de firmas e&nbsp;denominações</a>&nbsp;e faça a <a href="/Empresas/Pesquisar-Nomes-firmas-ou-denominacoes-existentes">pesquisa de nomes existentes</a>)<br>
 &nbsp;</li>
 <li>a morada da sede da entidade<br>
 &nbsp;</li>
 <li>o objeto social<br>
 &nbsp;</li>
 <li>a natureza jurídica da entidade.</li>
</ul>
</div>

<p>&nbsp;</p>

<h3 class="heading-regular">Como fazer o pedido online</h3>

<ol class="list--large-number">
 <li>Aceda ao serviço online&nbsp;</li>
 <li>Preencha o formulário</li>
 <li>Selecione o meio de pagamento (Multibanco ou cartão de crédito). Tem 5 dias seguidos a contar da data de submissão do pedido para fazer o pagamento.</li>
</ol>

<h2 class="heading-regular"><a id="Quando" name="Quando"></a>Quando pedir</h2>

<p>É necessário pedir o certificado de admissibilidade de nome para&nbsp;criar&nbsp;uma sociedade comercial, se pretender aprovar&nbsp;um&nbsp;nome para a sua empresa que não consta da lista de nomes pré-aprovados.</p>

<h3 class="heading-regular">Ao longo da vida da empresa, o certificado de admissibilidade é ainda necessário quando:<br>
&nbsp;</h3>

<div class="list list-checked">
<ul>
 <li>alterar o nome da empresa<br>
 &nbsp;</li>
 <li>mudar a&nbsp;sede para outro concelho, exceto se for uma&nbsp;sociedade comercial e a denominação for composta&nbsp;por expressão de fantasia, por exemplo, por&nbsp;nome da bolsa ou outra expressão sem significado<br>
 &nbsp;</li>
 <li>alterar as atividades a que a empresa se dedica (objeto social).</li>
</ul>
</div>

</div>
